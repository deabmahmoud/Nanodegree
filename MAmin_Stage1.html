What is World Wide Web (WWW)?

World Wide Web is a network of computers whose funtion is to serve webpages that we see on our browser. These web pages are made up of documents that are accessed via the internet using HTTP. These documents consists of HTML documents which may have reference to other documents like images, text, videos, or other links. Each webpage structures with boxes that contain different information like image, text, other boxes, video, etc.

What is HTML?

HTML (Hypertext Markup Language) is a standard language used to create webpages. Fundementals of any webpage are the elements. In most cases elements are made up of sets of tags, opening and closing tags. Although there are some elements that have no closing tags. Tags provide instruction for the web so that it knows what and when to present the content.

Basic structure of any html document consists of following main tags that are the base for any webpage.

What is CSS?

CSS (Cascading Style Sheet) is a language used to create presentation of a webpages. CSS can be included in a webpage by writing CSS in the <head> section of HTML, linking the CSS file in the HTML document, and writing style inline with HTML. Last option is not preferred since it may cause a lot of repeated code. With CSS, one can apply same style to a tag that appear multiple times in HTML code so that you are not repeating code to apply same style. Elements can inherit styles from thier parent.
